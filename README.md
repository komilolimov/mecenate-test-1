# Mecenate Feed App

Мобильное приложение для отображения ленты публикаций платформы Mecenate (аналог Patreon/Boosty). Разработано в рамках тестового задания.

## 🛠 Технологический стек

- **Платформа:** React Native + Expo (iOS / Android)
- **Язык:** TypeScript
- **Сетевой слой & Кэширование:** Axios + React Query (`@tanstack/react-query`)
- **Глобальное состояние:** MobX (`mobx`, `mobx-react-lite`)
- **UI & Стилизация:** Кастомный хук `useAppTheme` (Styles Factory + `useMemo`) для поддержки системной темной/светлой темы, SVG-иконки.
- **Архитектура:** Feature-Sliced Design (FSD) / Clean Architecture.

Реализованный функционал

[x] Лента постов: Отображение карточек с аватарами, текстом и обложками.

[x] Пагинация: Бесконечный скролл ленты (Infinite Queries).

[x] Фильтрация: Мгновенное переключение табов (Все / Бесплатные / Платные) через глобальный стор MobX.

[x] Платный контент: Заглушка с Blur-эффектом (expo-blur) для скрытия постов tier: "paid".

[x] Интерактив: Оптимистичное обновление UI (Optimistic Updates) при клике на лайк с использованием мутаций React Query.

[x] Обработка состояний: Кастомные UI-экраны загрузки, ошибки сети (с возможностью повтора) и пустого списка.

[x] Адаптивность: Полная поддержка переключения Dark/Light mode на уровне системы.
